Q. What is JavaScript?

Ans) JavaScript is the Programming Language for the Web. JavaScript can update and change both HTML and CSS. JavaScript can calculate, manipulate and validate data.



Q. Write down difference between Declaration and Initialization avariable.

Ans) Declaration tells the compiler about the existence of an entity in the program and its location. Whereas Initialization is the process of assigning a value to the Variable.



Q. In how many ways we can define a variable.

Ans) 4 Ways to Declare a JavaScript Variable: Using var, Using let, Using const & Using nothing



Q. Is variable is case sensitive?

Ans) Yes, JavaScript is a case-sensitive language



Q. When we declare a variable can we start it with any number?

Ans) No, JavaScript variable names should not start with a numeral (0-9).
